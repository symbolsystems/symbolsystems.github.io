CERT-MEM00_C (REC) Allocate and free memory in the same module, at the same level of abstraction
RULES

Do not allocate memory and expect that someone else will deallocate it later [CERT_C-MEM00-a-1]
Do not allocate memory and expect that someone else will deallocate it later [CERT_C-MEM00-b-1]
Do not allocate memory and expect that someone else will deallocate it later [CERT_C-MEM00-c-1]
Do not use resources that have been freed [CERT_C-MEM00-d-1]
Ensure resources are freed [CERT_C-MEM00-e-1]
