The library functions 'rewind', 'setbuf' and 'ctime' should not be used [CERT_C-ERR07-b-2]

DESCRIPTION 

"When you have a choice of two functions to accomplish the same task, prefer
the one with better error checking and reporting." [CERT C-ERR07]

This rule detects when the following C standard library functions are used:
'ctime', 'rewind' and 'setbuf'.

See also: MISRA2004-20_10
SINCE 

2023.2
BENEFITS 

The rule helps you prevent undefined behaviours.
EXAMPLE 

#include <time.h>
void f(){
    time_t * t1;
    ctime(t1);        // Violation
}
REPAIR 

#include <time.h>
void f(){
    time_t * t1;
    localtime(t1);    // OK
}
