CERT_C-DCL01 
CERT-DCL01_C (REC) Do not reuse variable names in subscopes
RULES

Identifier declared in a local or function prototype scope shall not hide an identifier declared in a global or namespace scope [CERT_C-DCL01-a-3]
Identifiers declared in an inner local scope should not hide identifiers declared in an outer local scope [CERT_C-DCL01-b-3]
