Do not allocate memory and expect that someone else will deallocate it later [CERT_C-MEM00-a-1]

DESCRIPTION 

If local memory in global function is allocated via new it
should be deleted in this function.
See also: MRM-20, MRM-19
BENEFITS 

This rule helps to ensure that all allocated memory
is being cleaned up properly.
EXAMPLE 

void foo( ) {
    int *var = new int( );  // Violation
}
REPAIR 

void foo( ) {
    int *var = new int( );  // OK
    delete var;
}
