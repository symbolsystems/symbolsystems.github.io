CERT_C-DCL11 
CERT-DCL11_C (REC) Understand the type issues associated with variadic functions
RULES

There should be no mismatch between the '%s' and '%c' format specifiers in the format string and their corresponding arguments in the invocation of a string formatting function [CERT_C-DCL11-a-2]
There should be no mismatch between the '%f' format specifier in the format string and its corresponding argument in the invocation of a string formatting function [CERT_C-DCL11-b-2]
There should be no mismatch between the '%i' and '%d' format specifiers in the string and their corresponding arguments in the invocation of a string formatting function [CERT_C-DCL11-c-2]
There should be no mismatch between the '%u' format specifier in the format string and its corresponding argument in the invocation of a string formatting function [CERT_C-DCL11-d-2]
There should be no mismatch between the '%p' format specifier in the format string and its corresponding argument in the invocation of a string formatting function [CERT_C-DCL11-e-2]
The number of format specifiers in the format string and the number of corresponding arguments in the invocation of a string formatting function should be equal [CERT_C-DCL11-f-2]
