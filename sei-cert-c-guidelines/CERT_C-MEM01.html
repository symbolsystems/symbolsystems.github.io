CERT_C-MEM01 
CERT-MEM01_C (REC) Store a new value in pointers immediately after free()
RULES

Do not use resources that have been freed [CERT_C-MEM01-a-2]
Always assign a new value to an expression that points to deallocated memory [CERT_C-MEM01-b-2]
Always assign a new value to global or member variable that points to deallocated memory [CERT_C-MEM01-c-2]
Always assign a new value to parameter or local variable that points to deallocated memory [CERT_C-MEM01-d-2]
