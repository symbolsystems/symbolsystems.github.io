CERT_C-ERR07 
CERT-ERR07_C (REC) Prefer functions that support error checking over equivalent functions that don't
RULES

The library functions atof, atoi and atol from library stdlib.h shall not be used [CERT_C-ERR07-a-2]
The library functions 'rewind', 'setbuf' and 'ctime' should not be used [CERT_C-ERR07-b-2]
