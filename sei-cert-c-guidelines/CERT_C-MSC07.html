CERT_C-MSC07 
CERT-MSC07_C (REC) Detect and remove dead code
RULES

There shall be no unreachable code in "else" block [CERT_C-MSC07-a-3]
There shall be no unreachable code after 'return', 'break', 'continue', 'goto', 'throw' statements, and after calls to functions with the 'noreturn' attribute [CERT_C-MSC07-b-3]
There shall be no unreachable code in "if/else/while/for" block [CERT_C-MSC07-c-3]
There shall be no unreachable code in switch statement [CERT_C-MSC07-d-3]
There shall be no unreachable code in 'for' loop [CERT_C-MSC07-e-3]
There shall be no unreachable code after 'if' or 'switch' statement [CERT_C-MSC07-f-3]
There shall be no unreachable code after "if" or "switch" statement inside while/for/do...while loop [CERT_C-MSC07-g-3]
Avoid switch with unreachable branches [CERT_C-MSC07-h-3]
Avoid unreachable methods [CERT_C-MSC07-i-3]
