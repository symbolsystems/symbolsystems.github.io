CERT-EXP20_C (REC) Perform explicit tests to determine success, true and false, and equality
RULES

Avoid comparing values with TRUE macro/enum constant using equality operators ("==", "!=") [CERT_C-EXP20-a-1]
Tests of a value against zero should be made explicit, unless the operand is effectively Boolean [CERT_C-EXP20-b-1]
