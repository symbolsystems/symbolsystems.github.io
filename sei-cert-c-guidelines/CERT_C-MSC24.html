CERT_C-MSC24 
CERT-MSC24_C (REC) Do not use deprecated or obsolescent functions
RULES

The library functions atof, atoi and atol from library stdlib.h shall not be used [CERT_C-MSC24-a-3]
The 'getenv()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [CERT_C-MSC24-b-3]
Avoid using unsafe string functions which may cause buffer overflows [CERT_C-MSC24-c-3]
Don't use unsafe C functions that do write to range-unchecked buffers [CERT_C-MSC24-d-3]
