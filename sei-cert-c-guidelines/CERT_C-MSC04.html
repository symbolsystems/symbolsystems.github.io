CERT_C-MSC04 
CERT-MSC04_C (REC) Use comments consistently and in a readable fashion
RULES

The character sequence /* shall not be used within a C-style comment [CERT_C-MSC04-a-3]
The character sequence // shall not be used within a C-style comment [CERT_C-MSC04-b-3]
The character sequence /* shall not be used within a C++-style comment [CERT_C-MSC04-c-3]
Line-splicing shall not be used in // comments [CERT_C-MSC04-d-3]
