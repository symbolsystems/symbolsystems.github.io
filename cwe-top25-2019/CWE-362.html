CWE-362 
CWE-362 Concurrent Execution using Shared Resource with Improper Synchronization ('Race Condition')
RULES

Usage of functions prone to race is not allowed [CWE-362-a-2]
Avoid race conditions while accessing files [CWE-362-b-2]
Use locks to prevent race conditions when modifying bit fields [CWE-362-c-3]
Avoid race conditions when using fork and file descriptors [CWE-362-d-1]
Do not use global variable with different locks set [CWE-362-e-1]
