No object or function identifier with static storage duration should be reused [MISRA2004-5_5_a-4]

DESCRIPTION 

Static object or function identifier shall not be reused for any other 
purpose within the program.
BENEFITS 

Reuse of static identifiers can lead to errors and confusion.
EXAMPLE 

static float a;
static void foo()
{
int a;    /* Violation */  
int foo;  /* Violation */
}
REPAIR 

Do not use static identifiers for any other purpose within the program.
REFERENCES 

1. ISO/DIS 26262
   point 8.4.4

2. MISRA-C:2004 Guidelines for the use of the C language in critical systems
   Chapter 6, Section 5
