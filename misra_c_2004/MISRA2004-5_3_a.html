Do not reuse typedef names [MISRA2004-5_3_a-3]

DESCRIPTION 

Typedef names shall not be reused.
BENEFITS 

Reuse of typedef names can lead to errors and confusion.
EXAMPLE 

typedef int MyInt;
void foo() 
{
  double MyInt;  /* Violation */
}
REPAIR 

typedef int MyInt;
void foo() 
{
  double MyVar;  /* OK */
}
REFERENCES 

1. MISRA-C:2004 Guidelines for the use of the C language in critical systems
   Chapter 6, Section 5

2. Origin: Misra Guidelines - Rule 17
