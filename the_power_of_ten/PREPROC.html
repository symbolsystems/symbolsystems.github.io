PREPROC 
Preprocessor
RULES

The #ifndef pre-processor directive will only be used to prevent multiple inclusions of the same header file [PREPROC-10-3]
The #endif pre-processor directives will only be used to prevent multiple inclusions of the same header file [PREPROC-10_b-3]
The #if pre-processor directive will only be used to prevent multiple inclusions of the same header file [PREPROC-10_c-3]
The #ifdef, #else, #elif preprocessor directives should not be used [PREPROC-11-3]
Macro should not contain token pasting, variable argument list nor recursive macro calls [PREPROC-13-3]
All macros must expand into complete syntactic units [PREPROC-14-3]
Pointer dereference operations may not be hidden in macro definitions [PREPROC-15-3]
#error directive shall not be used [PREPROC-22-3]
The #pragma directive shall not be used [PREPROC-23-3]
#undef shall not be used [PREPROC-25-3]
