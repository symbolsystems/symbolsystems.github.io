Do not call delete on non-pointers [PB-13-1]

DESCRIPTION 

This rule checks whether you call delete on non-pointer.
BENEFITS 

Prevents calling delete on non-pointer what results in an invalid operand.
EXAMPLE 

class Rhino {
public:
    Rhino( );
    Rhino( char* );
    Rhino( const char* );
    operator char*( );
};

void func()
{
    Rhino r;
    delete( r );  // Violation
}
REPAIR 

class Rhino {
public:
    Rhino( );
    Rhino( char* );
    Rhino( const char* );
    operator char*( );
};

void func()
{
    Rhino *r = new Rhino();
    delete r ;  // OK
}
