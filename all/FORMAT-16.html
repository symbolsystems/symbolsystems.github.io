There shall be a single ASCII space character preceding and following relational and equality operators [FORMAT-16-3]

DESCRIPTION 

This rule checks whether there is a single ASCII space character
preceding and following relational and equality operators.
BENEFITS 

This convention promotes readability by clearly separating
the operands from the operator.
EXAMPLE 

void foo(int x)
{
   if(x==1)     // Violation
   {
   }

   if(x  >  1)  // Violation
   {
   }

   if(x <=1)    // Violation
   {
   }
}
REPAIR 

void foo(int x)
{
   if(x == 1)   // OK
   {
   }

   if(x > 1)    // OK
   {
   }

   if(x <= 1)   // OK
   {
   }
}
