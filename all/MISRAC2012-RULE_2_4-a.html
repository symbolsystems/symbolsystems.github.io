A function should not contain unused local tag declarations [MISRAC2012-RULE_2_4-a-4]

DESCRIPTION 

"If a tag is declared but not used, then it is unclear to a reviewer if the tag 
is redundant or it has been left unused by mistake."[MISRA C:2012 Rule 2.4]

This rule detects cases when a function contains unused tag declarations.
SINCE 

v9.6
BENEFITS 

This rule improves legibility and increases efficiency of code.
EXAMPLE 

void foo ( void )
{
    typedef struct S1    /* Violation */
    {
        int i;
    } TS1;

    union U1             /* Violation */
    {
        int i;
    };

    enum E1              /* Violation */
    {
        E11, E12
    };

    TS1 ts1;
    ts1.i = E11 + E12;
}
REPAIR 

void foo ( void )
{
    typedef struct       /* OK */
    {
        int i;
    } TS1;

    union U1             /* OK */
    {
        int i;
    };

    enum                 /* OK */
    {
        E11, E12
    };

    union U1 u1;
    TS1 ts1;
    ts1.i = E11 + E12;
}
REFERENCES 

1. MISRA C:2012 Guidelines for the use of the C language in critical systems
   Section 8: Rules, Rule 2.4
