Begin all integer type variable with 'i' [NAMING-19-3]

DESCRIPTION 

All integer type variable should begin with 'i'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
  int Var;   // Violation
}
REPAIR 

void foo() 
{
  int iVar;  // OK
}
