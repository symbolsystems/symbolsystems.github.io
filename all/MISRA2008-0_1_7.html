The value returned by a function having a non-void return type that is not an overloaded operator shall always be used [MISRA2008-0_1_7-2]

DESCRIPTION 

"In C++ it is possible to call a function without using the return value,
which may be an error. The return value of a function shall always be used."
SINCE 

v7.2
EXCEPTIONS 

"The return value of a function may be discarded by use of a (void) cast.
Overloaded operators are excluded, as they should behave in the same way
as built-in operators."
BENEFITS 

Rule improves readability of code
EXAMPLE 

short func ( short para1 )
{
    return para1;
}

void discarded ( short para2 )
{
    func ( para2 );       // Violation
}
REPAIR 

short func ( short para1 )
{
    return para1;
}

void discarded ( short para2 )
{
    (void)func ( para2 );       // OK
}
