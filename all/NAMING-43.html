File name extension, if present, should be "*.cpp" or "*.h" [NAMING-43-3]

DESCRIPTION 

File name extension should be either:
  .cpp (recommended for implementation files)
  .h   (recommended for header files)

This rule reports a violation if file name extension does not match the 
templates above.

This rule does not report any violation if there is no extension at all
(no dot "." in file name).
SINCE 

v7.1
NOTES 

Rule is enabled for C++ only.
BENEFITS 

Rule improves readability and maintainability of code.
EXAMPLE 

//file.cc          // Violation
//file.hxx         // Violation
REPAIR 

//file.cpp        // OK   
//file.h          // OK
//file            // OK (no extension)
REFERENCES 

N/A
