Global prefixes should only be used for global variables [NAMING-04-3]

DESCRIPTION 

The legibility of code is hindered by using a global prefix to a local variable.
BENEFITS 

Makes code more legible.
EXAMPLE 

void foo()
{  
    int theVar = 0; // Violation
}
REPAIR 

int theVar = 0; // OK
