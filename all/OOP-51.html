Use namespace instead of class or structure containing only static functions [OOP-51-5]

DESCRIPTION 

Use namespace instead of class or structure containing only static functions.
Rule reports a violation if a non-template class contains only static functions
and does not inherit from any base class.
SINCE 

v7.3
BENEFITS 

Rule prevents creating unnecessary objects.
EXAMPLE 

class A{                     // Violation
    static void foo(){}
};

void foo(){
    A a;
    a.foo();
}
REPAIR 

namespace A{                 // OK
    void foo(){}
};

void foo(){
    A::foo();
}
