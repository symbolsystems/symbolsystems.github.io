Declare binary arithmetic and bitwise operators as non-members [HICPP-13_2_3-a-3]

DESCRIPTION 

Overloaded binary arithmetic and bitwise operators should be declared as
non-member functions.
SINCE 

v10.4
BENEFITS 

Allows operands of different types to be used.
EXAMPLE 

#include <iostream>

class A {
  public:
    bool operator*(A const &other);  // Violation
    bool operator==(A const &other); // OK
};
A operator+(int32_t lhs, A const &rhs);                // OK
A operator+(A const &lhs, int32_t rhs);                // OK
std::ostream &operator<<(std::ostream &o, A const &a); // OK
REPAIR 

#include <iostream>

class A {
  public:
    bool operator==(A const &other); // OK
};
bool operator*(A const &lhs, A const &other);          // OK
A operator+(int32_t lhs, A const &rhs);                // OK
A operator+(A const &lhs, int32_t rhs);                // OK
std::ostream &operator<<(std::ostream &o, A const &a); // OK
REFERENCES 

1. High Integrity C++ Coding Standard
   13.2.3. Declare binary arithmetic and bitwise operators as non-members
