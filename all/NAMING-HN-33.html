Hungarian notation for ostream type parameters and variables [NAMING-HN-33-3]

DESCRIPTION 

For ostream type parameters and variables use "os" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

#include <fstream>
using namespace std;

void foo(ostream &rOut)         // Violation
{
}
REPAIR 

#include <fstream>
using namespace std;

void foo(ostream &rosOut)      // OK
{
}
REFERENCES 

http://web.mst.edu/~cpp/common/hungarian.html
