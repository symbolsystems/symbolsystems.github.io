A value returned from a stdio character-oriented I/O function should not be implicitly converted to a type smaller than 'int' [CODSTA-146-3]

DESCRIPTION 

"When a primitive is cast to a smaller primitive, the high order bits
of the large value are lost in the conversion, potentially resulting
in an unexpected value that is not equal to the original value." [CWE-197]

The standard library character-oriented I/O functions return 'int' values
such as EOF, so their results should not be converted to smaller types.

The rule reports a violation if a value returned from the I/O function
is implicitly converted to a smaller type.

The rule checks values returned from:

- global functions:
    fgetc, getc, getchar, fputc, putc, putchar, ungetc

- member functions from class basic_istream or from classes that inherit
  from basic_istream:
    get, peek

- member functions from class basic_streambuf or from classes that inherit
  from basic_streambuf:
    snextc, sbumpc, sgetc, sputbackc, sungetc, sputc, underflow, uflow,
    pbackfail, overflow
SINCE 

v9.5
BENEFITS 

Following this rule prevents data loss.
EXAMPLE 

#include <stdio.h>
void foo1()
{
    FILE* pFile;
    char c;

    do {
        c = getc (pFile);    // Violation - implicit conversion from 'int'
                             // to 'char' type causes truncation
    } while (c != EOF);
}

#ifdef __cplusplus
#include <fstream>

void foo2 ()
{
    char str[256];
    std::ifstream is(str);
    char c;

    while (1)
    {
        c = is.get();        // Violation - implicit conversion from 'int'
                             // to 'char' type causes truncation
        if(c == -1)
            break;
    }
    is.close();
}
#endif
REPAIR 

#include <stdio.h>
void foo1()
{
    FILE* pFile;
    int c;

    do {
        c = getc (pFile);     // OK
    } while (c != EOF);
}

#ifdef __cplusplus
#include <fstream>

void foo2 ()
{
    char str[256];
    std::ifstream is(str);
    char c;

    while (1)
    {
        is.get(c);            // OK
        if(!is.good() )
            break;
    }
    is.close();
}
#endif
