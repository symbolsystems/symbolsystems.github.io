Physical lines should be less than eighty characters [FORMAT-04-3]

DESCRIPTION 

This rule checks that physical lines are less than eighty characters. This rule
promotes readability across various editors. Additionally, some printers will
not print columns greater than 80 columns. Tab is counted as one character.
BENEFITS 

Rule improves readability of code.
EXAMPLE 

const int MAXIMUM_LIMIT = 100;

void foo()
{
   int someLocalVariableHoldingTheLimit = (MAXIMUM_LIMIT * 10) - (MAXIMUM_LIMIT/2); // Violation
}
REPAIR 

const int MAXIMUM_LIMIT = 100;

void foo()
{
   int someLocalVariableHoldingTheLimit = (MAXIMUM_LIMIT * 10) -
                                          (MAXIMUM_LIMIT/2); // OK
}
