Hungarian notation for char pointer or reference types [NAMING-HN-10-3]

DESCRIPTION 

For char pointer or reference variables and parameters use "c" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

char* g_cmyVar;                                  // Violation
char* g_MyVariable;                              // Violation
static const char* g_skMyVariable;               // Violation
class A {
    char* m_MyVariable;                          // Violation
    static const char* m_skMyVariable;           // Violation
    void foo( char* MyParam ) {                  // Violation
        char* MyVariable;                        // Violation
        static const char* skMyVariable;         // Violation
    }
};
REPAIR 

char* g_cMyVar;                                  // OK
char* g_cMyVariable;                             // OK
static const char* g_skcMyVariable;              // OK
class A {
    char* m_cMyVariable;                         // OK
    static const char* m_skcMyVariable;          // OK
    void foo( char* cMyParam ) {                 // OK
        char* cMyVariable;                       // OK
        static const char* skcMyVariable;        // OK
    }
};
