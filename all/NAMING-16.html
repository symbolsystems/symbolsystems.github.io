Begin all float type variables with 'f' [NAMING-16-3]

DESCRIPTION 

All float type variables should begin with 'f'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
   float Var;	// Violation
}
REPAIR 

void foo() 
{
   float fVar;	// OK
}
REFERENCES 

Recommended by ParaSoft
