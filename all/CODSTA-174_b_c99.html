A program should not exceed the translation limits imposed by The Standard (c99) [CODSTA-174_b_c99-3]

DESCRIPTION 

"A program shall not exceed the translation limits imposed by the
implementation. The minimum translation limits are specified
by The Standard but an implementation may provide higher limits."
[MISRA C:2012 Rule 1.1]

This rule detects cases where the minimum translation limits specified
by the C99 standard are exceeded:
- 63 characters in a macro name 
- 4095 macro identifiers simultaneously defined in one translation unit 
  (this rule reports violations if more than 4095 macros are defined
   in one file)
- 4095 characters in a logical source line
- 4095 characters in a character string literal or wide string literal
- 63 nesting levels of conditional inclusion
- 63 nesting levels of parenthesized expressions or declarators
- 127 parameters in one macro definition 
- 127 arguments in one macro invocation
   (this rule reports violations only on invocations of macros that are
   defined in the same file)

See also: CODSTA-174_a_c99
SINCE 

v9.6
NOTES 

You can use the Rule Wizard to change the default limits, if necessary.
These limits are defined in the Python method 'defineLimits'.
For example, if you need to increase the limit for the initial characters
in a macro name to 127, change the following definition:

  # characters in a macro name 
  LimitTxt_MacroNameLength = 63

to:

  # characters in a macro name 
  LimitTxt_MacroNameLength = 127
BENEFITS 

This rule prevents undefined behaviors and improves readability of code.
EXAMPLE 

/* More then 63 characters in a macro name */
#define MAC_64_CHARACTERS_IN_NAME_01234  B  /* Violation */

/* More than 4095 macro identifiers defined in the current file */
/* Violation will be reported at the begin of file */
#define M1    1
/* ... */
#define M4096 4096

/* More than 63 nesting levels of conditional inclusion */
#ifdef  NAME1
#ifdef  NAME2
#ifdef  NAME3
/*....*/
#ifdef  NAME63  
#ifdef  NAME64  /* Violation */
...
#endif
#endif
/*....*/
#endif
#endif
#endif

/* More than 127 parameters in one macro definition */
#define ADD32(X1,X2,/* ... */ X63,/* ... */ X128) (X1*X128)     /* Violation */

void foo(){
    /* More than 127 arguments in one macro invocation */
    ADD32(1,2,/* ... */ 127,128);                               /* Violation */
}

void foo2(int n)
{
    /* More than 4095 characters in a character string literal
       or wide string literal */
    char* str = "123456789... 4080 chars in this line ...7890"  /* Violation */
                "1234567890123456";

    /* More than 63 nesting levels of parenthesized declarators */
    int (((((((((( (((((((((( ((((((((((                        /* Violation */
        (((((((((( (((((((((( (((((((((( 
        ((((( i )))))
        )))))))))) )))))))))) ))))))))))
        )))))))))) )))))))))) ))))))))));

    /* More than 63 nesting levels of parenthesized expression */
    i = (((((((((( (((((((((( ((((((((((                        /* Violation */
        (((((((((( (((((((((( (((((((((( 
        ((((( 10 * n )))))
        )))))))))) )))))))))) ))))))))))
        )))))))))) )))))))))) ))))))))));
}
REPAIR 

Do not exceed translation limits of an implementation.
