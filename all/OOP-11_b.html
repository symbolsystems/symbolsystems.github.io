Friend declarations shall not be used except declarations of comparison operators [OOP-11_b-3]

DESCRIPTION 

"Friend declarations reduce encapsulation and result in code that is more
difficult to maintain.
It is allowed to declare comparison operators as friend functions."
[AUTOSAR-A11-3-1]

This rule detects friend declarations other than declarations of comparison
operators.
SINCE 

2022.1
BENEFITS 

This rule helps you avoid using friend declarations and improve code
maintainability.
EXAMPLE 

int foo( );
class C {
    friend int foo( );                // Violation
};

class D {
friend class C;                       // Violation
};
REPAIR 

// example of correct code
int foo( );
class C {};                           // OK

class D {
public:
    friend bool operator==(D const& l, D const& r);      // OK
};
REFERENCES 

1. AUTOSAR C++14 Coding Guidelines
   A11-3-1. Friend declarations shall not be used
