Initialize all pointer variables [INIT-04-2]

DESCRIPTION 

This rule checks if all of your pointer variables are initialized.
NOTES 

This rule enforces a good coding standard practice: always initialize pointer 
variables, even if you're going to change (assign) the variable  at once.
BENEFITS 

Initializing pointer variables prevents dereferencing of uninitialized pointers.
EXAMPLE 

void foo(int y, int * p)
{
   int *i;  // Violation
   int *j;  // Violation
   if (y) {
      j = 0;
   } else {
      j = p;
   }
}
REPAIR 

void foo(int y, int * p)
{
   int *i = 0;            // OK
   int *j = y ? 0 : p;  // OK
}
