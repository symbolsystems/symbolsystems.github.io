Do not create inline non-member functions that contain local static data [PB-19-3]

DESCRIPTION 

"If you create an inline non-member function containing a local static object, 
you may end up with more than one copy of the static object in your program!"
SINCE 

v7.0
BENEFITS 

Rule prevents from declaring inline local variable inside inline function.
EXAMPLE 

inline int increase( ) {   // Violation
    static int count = 0;
    count++;
    return count;
}
REPAIR 

int increase( ) {          // OK
    static int count = 0;
    count++;
    return count;
}
REFERENCES 

Scott Meyers, "More Effective C++: 35 New Ways to Improve 
Your Programs and Designs", Addison-Wesley, Copyright 1996, 
Chapter: "Techniques", Item 26
