Hungarian notation for double-precision floating point types [NAMING-HN-12-3]

DESCRIPTION 

For double-precision floating point variables and parameters use "d" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

double g_dmyVar;                                  // Violation
double g_MyVariable;                              // Violation
static const double g_skMyVariable = 0.0;         // Violation
class B {
    double m_MyVariable;                          // Violation
    static const double m_skMyVariable;           // Violation
    void foo( double MyParam ) {                  // Violation
        double MyVariable;                        // Violation
        static const double skMyVariable = 0.0;   // Violation
    }
};
REPAIR 

double g_dMyVar;                                  // OK
double g_dMyVariable;                             // OK
static const double g_skdMyVariable = 0.0;        // OK
class A {
    double m_dMyVariable;                         // OK
    static const double m_skdMyVariable;          // OK
    void foo( double dMyParam ) {                 // OK
        double dMyVariable;                       // OK
        static const double skdMyVariable = 0.0;  // OK
    }
};
