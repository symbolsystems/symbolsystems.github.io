Hungarian notation for member variables [NAMING-HN-29-3]

DESCRIPTION 

For member variables use "m_" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

class CSample
{
    int iSize;     // Violation
};
REPAIR 

class CSample
{
    int m_iSize;   // OK
};
REFERENCES 

http://web.mst.edu/~cpp/common/hungarian.html
