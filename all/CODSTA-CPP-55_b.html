Private and protected methods shall not declare parameters with array type [CODSTA-CPP-55_b-2]

DESCRIPTION 

Array parameters should not be used in private nor in protected methods.
Instead, the Array class should be used.

See also: CODSTA-142, CODSTA-CPP-55
SINCE 

v9.5
NOTES 

Rule defines an interface as a private or protected method.
BENEFITS 

Arrays degenerate to pointers when passed as parameters.
This "array decay" problem has long been known to be a source of errors.
EXAMPLE 

class Sample
{
  private:
    void foo(int a[]); // Violation
};
REPAIR 

class Array
{
  /* Array implementation*/
};


class Sample
{
  private:
    void goo(Array a); // OK
};
