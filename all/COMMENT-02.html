Provide copyright information [COMMENT-02-3]

DESCRIPTION 

"Every source file will be documented with an introductory comment
that provides information on the copyright"

The rule requires that before the first use of source code in file
should be a comment containing the 'copyright' word (case insensitive).
It is allowed to place characters different that letters and numbers
directly before or after the 'copyright' word (e.g. '*copyright:' is correct,
'copyright2' is not correct).
The rule checks /* */ and // comments.
BENEFITS 

The rule improves readability and maintainability.
EXAMPLE 

// Violation - no information

void foo();

// Copyright should be placed in a comment before first source code in file.
REPAIR 

// Copyright

void foo();
