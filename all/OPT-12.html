If a file-level static variable is used/referenced in one class only then include that variable in the class itself [OPT-12-3]

DESCRIPTION 

If a file-level static variable is used/referenced in one class only,
it is better to include that variable in the class itself.
See also: OPT-11
SINCE 

v7.0
BENEFITS 

Memory optimization and clarity of intent expression/maintainability.
EXAMPLE 

static int s;               // Violation
const static int cs = 0;    // Violation

class A
{
    void foo1(int i) {
        s = i + cs;
    }
    void foo2(int j ) {
        s = j + cs;
    }
};
REPAIR 

class A
{
    static int s;               // OK
    const static int cs = 0;    // OK
    void foo1(int i) {
        s = i + cs;
    }
    void foo2(int j ) {
        s = j + cs;
    }
};
