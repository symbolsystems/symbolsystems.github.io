A virtual base shall be explicitly declared in each derived class [OOP-39-4]

DESCRIPTION 

A virtual base shall be explicitly declared in each derived class.

See also: OOP-39_b
SINCE 

v7.1
BENEFITS 

Documentation purposes.
EXAMPLE 

class AA {};
class BB : virtual AA {};
class CC : virtual AA {};
class DD : BB{};          // Violation
class EE : CC{};          // Violation
class FF : DD{};          // Violation
REPAIR 

class AA {};
class BB : virtual AA {};
class CC : virtual AA {};
class DD : BB, virtual AA {};     // OK
class EE : CC, virtual AA {};     // OK
class FF : DD, EE, virtual AA {}; // OK
REFERENCES 

N/A
