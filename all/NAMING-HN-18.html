Hungarian notation for floating point pointer, array, or reference types [NAMING-HN-18-3]

DESCRIPTION 

For floating point pointer, array, or reference 
variables and parameters use "f" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

float* g_pfmyVar;                                  // Violation
float* g_pMyVariable;                              // Violation
static const float* g_spkMyVariable;               // Violation
class B {
    float* m_pMyVariable;                          // Violation
    static const float* m_spkMyVariable;           // Violation
    void foo( float* pMyParam ) {                  // Violation
        float* pMyVariable;                        // Violation
        static const float* spkMyVariable;         // Violation
    }
};
REPAIR 

float* g_pfMyVar;                                  // OK
float* g_pfMyVariable;                             // OK
static const float* g_spkfMyVariable;              // OK
class A {
    float* m_pfMyVariable;                         // OK
    static const float* m_spkfMyVariable;          // OK
    void foo( float* pfMyParam ) {                 // OK
        float* pfMyVariable;                       // OK
        static const float* spkfMyVariable;        // OK
    }
};
