Begin all short integer variables with 'si' [NAMING-23-3]

DESCRIPTION 

All short integer variables should begin with 'si'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
  short int Var;    // Violation
}
REPAIR 

void foo() 
{
  short int siVar;  // OK
}
