A relational operator shall return a boolean value [HICPP-13_2_2-a-3]

DESCRIPTION 

"Returning a type 'bool' from a relational operator is consistent with the C++
Standard Library." [AUTOSAR A13-2-3]

This rule detects when the return type of a function overloading
a relational operator is not of type bool.
SINCE 

v10.4
BENEFITS 

This rule improves code readability and consistency with the C++ standard
library. The rule also helps you achieve compliance with some coding
guidelines.
EXAMPLE 

class A
{};

int operator>(A const&, A const&) noexcept // Violation
{
    return -1;
}
REPAIR 

class A
{};

bool operator>(A const&, A const&) noexcept // OK
{
    return false;
}
REFERENCES 

1. AUTOSAR C++14 Coding Guidelines
   A13-2-3. A relational operator shall return a boolean value

2. High Integrity C++ Coding Standard
   13.2.2. Ensure that the return type of an overloaded binary operator
   matches the built-in counterparts
