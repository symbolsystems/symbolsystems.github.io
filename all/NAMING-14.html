Begin class data member names with 'its' [NAMING-14-3]

DESCRIPTION 

Class data member names should begin with 'its'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

class Foo {
private:
	int bar;	// Violation
};
REPAIR 

class Foo {
private:
	int itsBar;	// OK
};
REFERENCES
