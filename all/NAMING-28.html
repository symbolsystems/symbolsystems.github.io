Use lowercase letters for structure and union member names [NAMING-28-3]

DESCRIPTION 

All structure and union members shall be lowercase proper. This convention will 
make the recognition of structure/union members easier.
BENEFITS 

Makes code more readable
EXAMPLE 

struct date_foo2 { 
    int datemonth;
    int DateDay;    // Violation
    int Dateyear;   // Violation  
};
REPAIR 

struct date_foo1 {
    int dateMonth;
    int dateDay;    // OK
    int dateYear;   // OK
};
REFERENCES 

Motorola Coding Standards R-19
