In an enumerator list, the "=" construct shall not be used to explicitly initialise members other than the first, unless all items are explicitly initialised [CERT_C-INT09-a-3]

DESCRIPTION 

"In enumerator list, the '=' construct shall not be used 
to explicitly initialize members other than the first,
unless all items are explicitly initialized."
BENEFITS 

Helps avoid errors and confusion.
EXAMPLE 

enum TEST { /* Violation */
    X = 1,
    Y,
    Z = 3,
};

enum TEST2 { /* Violation */
    X2,
    Y2 = 2,
    Z2,
};

enum TEST3 { /* Violation */
    X3,
    Y3,
    Z3 = 3,
};
REPAIR 

enum TEST { /* OK */
    X,
    Y,
    Z,
};

enum TEST2 { /* OK */
    X2 = 1,
    Y2,
    Z2,
};

enum TEST3 { /* OK */
    X3 = 1,
    Y3 = 2,
    Z3 = 3,
};
REFERENCES

4. MISRA-C:2004 Guidelines for the use of the C language in critical systems
   Chapter 6, Section 9
