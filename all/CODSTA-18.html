Prefer 'while' statements over 'do' statements [CODSTA-18-5]

DESCRIPTION 

This rule detects 'do-while' statements.
BENEFITS 

This rule helps you improve readability and prevents bug-prone code.
EXAMPLE 

#define DO_WHILE(arg)       \
  do /* OK */               \
  {                         \
     /*...*/                \
  } while (arg == 0)

void fun1()
{
    int i = 0;
    do {                // Violation
        // do something
        i++;
    } while (i < 20);
    DO_WHILE(i);        // Violation
}
REPAIR 

void fun1()
{
    int i = 1;
    // do something
    while(i < 20) {     // OK
        // do something
        i++;
    }
}
