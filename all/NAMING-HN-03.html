Hungarian notation for bool pointer, array, or reference types [NAMING-HN-03-3]

DESCRIPTION 

For bool pointer, array, or reference variables and parameters use "b" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

bool* g_pbmyVar;                                  // Violation
bool* g_pMyVariable;                              // Violation
static const bool* g_spkMyVariable;               // Violation
class A {
    bool* m_pMyVariable;                          // Violation
    static const bool* m_spkMyVariable;           // Violation
    void foo( bool* pMyParam ) {                  // Violation
        bool* pMyVariable;                        // Violation
        static const bool* spkMyVariable;         // Violation
    }
};
REPAIR 

bool* g_pbMyVar;                                  // OK
bool* g_pbMyVariable;                             // OK
static const bool* g_spkbMyVariable;              // OK
class A {
    bool* m_pbMyVariable;                         // OK
    static const bool* m_spkbMyVariable;          // OK
    void foo( bool* pbMyParam ) {                 // OK
        bool* pbMyVariable;                       // OK
        static const bool* spkbMyVariable;        // OK
    }
};
