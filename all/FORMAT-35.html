When declaring functions, the leading parenthesis and the first argument are to be written on the same line as the function name [FORMAT-35-3]

DESCRIPTION 

"When declaring functions, the leading parenthesis and the first argument
(if any) are to be written on the same line as the function name."
BENEFITS 

Rule improves readability and maintainability of code.
EXAMPLE 

void foo
( );            // Violation

void goo(       // Violation
int a );
REPAIR 

void foo( );        // OK

void goo( int a );  // OK
REFERENCES 

1. [Ellemtel 92] "Programming in C++: Rules and Recommendations"
   Ellemtel Telecommunication Systems Laboratories, Sweden.
