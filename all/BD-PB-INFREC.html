Avoid infinite recursion [BD-PB-INFREC-2]

DESCRIPTION 
This rule detects infinitely recursive methods. A method is infinitely recursive when every
path that results from the method being entered leads to a recursive call. This rule also
detects indirect recursion.
SINCE 
v10.1
NOTES 
N/A
SECURITY RELEVANCE 
N/A
PARAMETERS 
N/A
BENEFITS 
N/A
DRAWBACKS 
N/A
EXAMPLE 
Here is an example that triggers a violation:

int binarySearch(int* tab, int a, int b, int val)
{
    int index = (a + b) / 2;
    if (tab[index] > val) {
        return binarySearch(tab, a, index - 1, val); // VIOLATION
    }
    return binarySearch(tab, index + 1, b, val); // VIOLATION
}
REPAIR 

int binarySearch(int* tab, int a, int b, int val)
{
    int index = (a + b) / 2;
    if (tab[index] > val) {
        return binarySearch(tab, a, index - 1, val);
    } else if (tab[index] < val) {
        return binarySearch(tab, index + 1, b, val);
    } else {
        return index;
    }
} // NO VIOLATION
