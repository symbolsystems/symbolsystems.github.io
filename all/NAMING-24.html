Begin all signed character variables with 'c' [NAMING-24-3]

DESCRIPTION 

All signed character variables should begin with 'c'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
    signed char Var;	// Violation
}
REPAIR 

void foo() 
{
   signed char cVar;	// OK
}
REFERENCES 

Recommended by ParaSoft
