Braces "{}" which enclose a block should be placed in the same column [FORMAT-34-3]

DESCRIPTION 

Rule checks if opening and closing braces are placed in the same column.
See also: FORMAT-02, FORMAT-03, FORMAT-42, FORMAT-43, FORMAT-44
EXCEPTIONS 

Rule does not adhere to enums, initializations and empty blocks.
enum E {EN1 = 2, EN2 = 1};  // OK
int array[] = {1,2,3,4,5};  // OK
class FooBar{};             // OK
Rule does not report violation if in line with brace 'tab' is used.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

class Foobar {   // Violation
   int a;
};

struct S1 {      // Violation
    int a;
};
REPAIR 

class Foobar 
{                // OK
    int a;
};

struct S1
{                // OK
    int a;
};
