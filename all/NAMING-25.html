Begin all terminated characters string variables with 'sz' [NAMING-25-3]

DESCRIPTION 

All terminated characters string variables should begin with 'sz'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
  char Var[] = "\n"; // Violation
}
REPAIR 

void foo() 
{
  char szVar[] = "\n";  // OK
}
