The names of structures shall begin with an uppercase letter and contain a suffix '_t' at the end [NAMING-12-3]

DESCRIPTION 

The rule reports a violation if the name of struct does not begin
with an uppercase letter or is not followed by a suffix '_t'.
See also: NAMING-09, NAMING-10, NAMING-11
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

struct Str1    // Violation
{ 
   int i;
};     
struct str2_t  // Violation
{ 
   int i;
};
REPAIR 

struct Str1_t    // OK
{
   int i;
};
struct Str2_t    // OK
{
   int i;
};
