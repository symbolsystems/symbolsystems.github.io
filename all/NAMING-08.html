Begin all boolean type variables with 'b' [NAMING-08-3]

DESCRIPTION 

Begin all boolean type variables with 'b'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
    bool Var; // Violation
}
REPAIR 

void foo() 
{
    bool bVar; // OK
}

Recommended by ParaSoft
