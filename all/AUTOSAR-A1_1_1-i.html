Do not use the increment operator (++) on an operand of type 'bool' [AUTOSAR-A1_1_1-i-2]

DESCRIPTION 

"All code shall conform to ISO/IEC 14882:2014 - Programming Language C++ and
shall not use deprecated features." [AUTOSAR-A1-1-1]

"Incrementing an object with bool type results in its value been set to true.
This feature was deprecated in the 1998 C++ Language Standard and thus may be
withdrawn in a later version." [High Integrity C++]

The rule detects when the increment operator (++) has an operand of type
'bool'.
SINCE 

v10.4
BENEFITS 

This rule helps you ensure that your code complies with the up-to-date version
of the C++ Language Standard. This allows you to increase longevity of
an application and address the defects and vulnerabilities that may emerge
as deployment environments evolve.
EXAMPLE 

void foo(bool bp)
{
    bool bv;
    bv++;                   // Violation
    ++bp;                   // Violation
}
REPAIR 

You can fix the code by explicitly assigning the value 'true'
to the bool type.

void foo(bool bp)
{
    bp = true;              // OK
}
