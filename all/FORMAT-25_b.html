Parenthesis shall be used with the "return" statement [FORMAT-25_b-3]

DESCRIPTION 

This rule checks whether parentheses are used with the "return" statement.
See also: FORMAT-23, FORMAT-24
BENEFITS 

Improves the readability of code.
EXAMPLE 

#define true 1
#define false 0
int foo() 
{
   return true; // Violation
}
REPAIR 

#define true 1
#define false 0
int goo() 
{
   return(true); // OK
}

void xoo() 
{
   return;      // OK
}
