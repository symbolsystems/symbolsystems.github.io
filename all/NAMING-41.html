Header files will always have a file name extension of '.h' [NAMING-41-3]

DESCRIPTION 

The rule reports violations on header files which does not have '.h' extension.
See also: PREPROC-08, NAMING-37, NAMING-38
SINCE 

v7.1
NOTES 

A header file is defined as any file that is included via #include.
BENEFITS 

Rule makes source code more readable.
EXAMPLE 

// file.cpp
#include "file.hpp"

// file.hpp                 // Violation
extern int i;
REPAIR 

// file.cpp
#include "file.h"

// file.h                   // OK
extern int i;
REFERENCES 

JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
