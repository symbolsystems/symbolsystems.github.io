A virtual function shall only be overridden by a pure virtual function if it is itself declared as pure virtual [OOP-43-4]

DESCRIPTION 

"A virtual function shall only be overridden by a pure
virtual function if it is itself declared as pure virtual."
SINCE 

v7.2
BENEFITS 

Rule prevents writing a code contrary to developer expectations.
EXAMPLE 

class A
{
    public:
    virtual void foo ( );
};
class B: public A
{
    public:
    virtual void foo ( ) = 0; // Violation
};
REPAIR 

Do not override non-pure virtual functions by pure virtual functions
REFERENCES 

1. AUTOSAR C++14 Coding Guidelines
   M10-3-3. A virtual function shall only be overridden by a pure virtual
   function if it is itself declared as pure virtual

2. MISRA C++:2008 Guidelines for the use of the C++ language in critical systems
   Chapter 6, Section 10, Rule 10-3-3
