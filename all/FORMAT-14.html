There shall be a single ASCII space character preceding ternary conditional operator [FORMAT-14-3]

DESCRIPTION 

This rule checks that there is a single ASCII space character
preceding ternary conditional operator.
See also: FORMAT-15
BENEFITS 

This convention promotes readability by clearly separating
the operands from the operator.
EXAMPLE 

void foo() 
{
    int j=10;
    int i=1;
    j = ( i < 0 )  ?  ( -i ) : ( i );  // Violation
    j = ( i < 0 )?( -i ) : ( i );      // Violation
}
REPAIR 

void foo() 
{
    int j=10;
    int i=1;
    j = ( i < 0 ) ? ( -i ) : ( i );  // OK
}
