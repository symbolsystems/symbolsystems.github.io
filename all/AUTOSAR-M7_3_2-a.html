The identifier main shall not be used for a function other than the global function main [AUTOSAR-M7_3_2-a-2]

DESCRIPTION 

"main (or its equivalent) is usually the entry point to the program and is the
only identifier which must be in the global namespace. The use of main for
other functions may not meet developer expectations."
SINCE 

v7.2
BENEFITS 

Rule improves readability of code.
EXAMPLE 

namespace
{
    int main ( ){} // Violation
}
REPAIR 

int main ( ){}     // OK
