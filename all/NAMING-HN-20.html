Hungarian notation for structs declaration [NAMING-HN-20-3]

DESCRIPTION 

For structs declaration use "S" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

struct Figure  // Violation
{
};
REPAIR 

struct SFigure  // OK
{
};
REFERENCES 

http://web.mst.edu/~cpp/common/hungarian.html
