Do not use operator ++ or -- in the conditional expression of if, while, or switch [CODSTA-33-3]

DESCRIPTION 

Using the operator ++ or -- in the conditional part of if, while or switch can 
make code difficult-to-read and error-prone.
BENEFITS 

Makes code more readable.
EXAMPLE 

int Foo (int iVar)
{	
    if (iVar-- && iVar<10)  // Violation 
    {
        iVar += 10;
    } 
    else 
    {
        iVar -= 10;
    }
    return iVar;
}
REPAIR 

int Foo (int iVar)
{	
    if (iVar<10)  // OK
    {
        iVar += 10;
    } 
    else 
    {
        iVar -= 10;
    }
    return iVar;
}
