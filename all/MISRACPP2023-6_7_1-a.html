Local variables shall not have static storage duration [MISRACPP2023-6_7_1-a-2]

DESCRIPTION 

Functions containing variables with static storage duration are more difficult
to understand and maintain. Using non-const variables with static storage
duration, even if they have no linkage, can result in hidden temporal coupling.

This rule detects declarations of non-const static local variables.
BENEFITS 

This rule helps you improve maintainability of the code and prevent data races.
EXAMPLE 

void f1()
{
    static int x1 = 0;        // Violation
    x1++;
}
REPAIR 

static int x1 = 0;            // OK
void f1()
{
    x1++;
}

void f2()
{
    static const int s1 = 0;  // OK
}
