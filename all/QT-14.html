If you use Q_DECLARE_FLAGS, you must also use Q_DECLARE_OPERATORS_FOR_FLAGS [QT-14-3]

DESCRIPTION 

If you use Q_DECLARE_FLAGS, you must also use 
Q_DECLARE_OPERATORS_FOR_FLAGS.
BENEFITS 

Q_DECLARE_FLAGS creates a type-safe way of storing OR-combinations of enum
values. Q_DECLARE_OPERATORS_FOR_FLAGS declares a global operator|() for these
type-safe flags, so they may be used outside the scope of where the flags
are declared.
EXAMPLE 

#include <QtGui/QtGui>

enum Value { FirstVal, SecondVal, ThirdVal };

Q_DECLARE_FLAGS(Values,Value)                    // Violation
REPAIR 

#include <QtGui/QtGui>

enum Value { FirstVal, SecondVal, ThirdVal };

Q_DECLARE_FLAGS(Values,Value)                    // OK
Q_DECLARE_OPERATORS_FOR_FLAGS(Values)
