Hungarian notation for array variables and parameters [NAMING-HN-01-3]

DESCRIPTION 

For array (stands for range) variables and parameters use "rg" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

const int kiMAX=10;

float fTmp[kiMAX];        // Violation

void foo()
{
    int iTab[10];         // Violation
    int iTab2[kiMAX];     // Violation
}
REPAIR 

const int kiMAX=10;

float rgfTmp[kiMAX];      // OK

void foo()
{
    int rgiTab[10];       // OK
    int rgiTab2[kiMAX];   // OK
}
