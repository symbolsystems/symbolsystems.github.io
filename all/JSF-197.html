Floating point variables shall not be used as loop counters [JSF-197-2]

DESCRIPTION 

Floating point variables shall not be used as loop counters.
BENEFITS 

Prevents giving unexpected results when the variable is tested.
EXAMPLE 

void foo() 
{
  float i;
  for (i=0.;i<10.;i++)  /* Violation */
  { 
  }
}
REPAIR 

void foo() 
{
  int i;
  for (i=0;i<10;i++) /* OK */
  {  
  }
}
