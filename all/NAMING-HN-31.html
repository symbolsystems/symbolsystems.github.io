Hungarian notation for int pointer, array, or reference types [NAMING-HN-31-3]

DESCRIPTION 

For int pointer, array, or reference variables and parameters use "n" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

int* g_pnmyVar;                                  // Violation
int* g_pMyVariable;                              // Violation
static const int* g_spkMyVariable;               // Violation
class A {
    int* m_pMyVariable;                          // Violation
    static const int* m_spkMyVariable;           // Violation
    void foo( int* pMyParam ) {                  // Violation
        int* pMyVariable;                        // Violation
        static const int* spkMyVariable;         // Violation
    }
};
REPAIR 

int* g_pnMyVar;                                 // OK
int* g_pnMyVariable;                            // OK
static const int* g_spknMyVariable;             // OK
class A {
    int* m_pnMyVariable;                        // OK
    static const int* m_spknMyVariable;         // OK
    void foo( int* pnMyParam ) {                // OK
        int* pnMyVariable;                      // OK
        static const int* spknMyVariable;       // OK
    }
};
