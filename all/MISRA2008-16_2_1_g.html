The #pragma directive shall not be used [MISRA2008-16_2_1_g-2]

DESCRIPTION 

"The #pragma directive is implementation-defined and causes the implementation
to behave in implementation-defined manner" [AUTOSAR-16-7-1]

The rule detects '#pragma' pre-processor directives.

See also: PREPROC-11, PREPROC-22, MISRA2004-3_4
SINCE 

v10.4
BENEFITS 

This rule helps you prevent implementation-defined behavior.
EXAMPLE 

#pragma once               // Violation
REPAIR 

#ifndef FOO_H              // OK
#define FOO_H
/* ... */
#endif

   (6) #else, (7) #define, (8) #endif, (9) #include
