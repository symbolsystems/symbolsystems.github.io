Functions that begin with 'is' should return boolean values [NAMING-20-3]

DESCRIPTION 

Rule checks if a function that begins with 'is_' or 'is[A-Z]' return a boolean
value.
NOTES 

As boolean types are recognized typedefs which have names
begin with 'bool' (ignoring case)

The rule doesn't check coroutine functions.
BENEFITS 

Improvement of legibility and clarifies what the return value means.
EXAMPLE 

int  isPositive(int x)	 // Violation
{        
    return x > 0;
}
REPAIR 

bool isPositive(int x)   // OK
{  
    return x > 0;
}
