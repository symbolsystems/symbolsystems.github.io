Hungarian notation for floating point types [NAMING-HN-17-3]

DESCRIPTION 

For floating point variables and parameters use "f" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

float g_fmyVar;                                  // Violation
float g_MyVariable;                              // Violation
static const float g_skMyVariable = 0.0;         // Violation
class B {
    float m_MyVariable;                          // Violation
    static const float m_skMyVariable;           // Violation
    void foo( float MyParam ) {                  // Violation
        float MyVariable;                        // Violation
        static const float skMyVariable = 0.0;   // Violation
    }
};
REPAIR 

float g_fMyVar;                                  // OK
float g_fMyVariable;                             // OK
static const float g_skfMyVariable = 0.0;        // OK
class A {
    float m_fMyVariable;                         // OK
    static const float m_skfMyVariable;          // OK
    void foo( float fMyParam ) {                 // OK
        float fMyVariable;                       // OK
        static const float skfMyVariable = 0.0;  // OK
    }
};
REFERENCES
