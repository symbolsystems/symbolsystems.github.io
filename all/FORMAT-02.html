Place an opening brace '{' on its own line [FORMAT-02-3]

DESCRIPTION 

The rule reports a violation if an opening brace '{' is not placed
on separate line.
See also: FORMAT-03, FORMAT-34, FORMAT-42, FORMAT-43, FORMAT-44
EXCEPTIONS 

Rule does not adhere to enums, initializations and empty blocks.
enum E {EN1 = 2, EN2 = 1};  // OK
int array[] = {1,2,3,4,5};  // OK
class FooBar{};             // OK
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

class A1
{ int a;                      // Violation
};

struct S1 {                   // Violation
    int a;
};
REPAIR 

class A1
{
    int a;                    // OK
};

struct S1
{                             // OK
    int a;
};
