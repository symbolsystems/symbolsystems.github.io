Do not use platform-specific multi-threading facilities [HICPP-18_1_1-a-3]

DESCRIPTION 

Use multi-threading facilities from the C++ standard library to reduce code
maintenance and improve portability.
SINCE 

v10.4
BENEFITS 

This rule reduces code maintenance and improves portability.
EXAMPLE 

#include <pthread.h> // Violation
void *thread1(void *);
void f1() {
    pthread_t t1;
    pthread_create(&t1, nullptr, thread1, 0);
    // ...
}
REPAIR 

#include <thread> // OK
void thread2();
void f2() {
    std::thread t1(thread2);
    // ...
}
REFERENCES 

1. High Integrity C++ Coding Standard
   18.1.1. Do not use platform specific multi-threading facilities
