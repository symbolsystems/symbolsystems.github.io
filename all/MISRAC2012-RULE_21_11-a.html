The standard header file <tgmath.h> shall not be used [MISRAC2012-RULE_21_11-a-2]

DESCRIPTION 

"The standard header file <tgmath.h> should not be #included."
[MISRA C:2012 Rule 21.11 with Amendment 3]

The rule reports a violation message if a file includes any of the following
headers: 'tgmath.h' or 'ctgmath'.
SINCE 

v9.5
BENEFITS 

The rule prevents undefined behaviours.
EXAMPLE 

#include <tgmath.h>    /* Violation */

double d;

void f ( void )
{
    sqrt ( d );  /* generic sqrt used */
}
REPAIR 

#include <math.h>      /* OK */

double d;

void f ( void )
{
    sqrt ( d ); /* math.h version used */
}
