Prefer "a @= b" than "a = a @ b", where "@" is +, -, *, /, % [OPT-07-3]

DESCRIPTION 

Prefer "a @= b" than "a = a @ b", where "@" is +, -, *, /, %.
See also: OPT-08
BENEFITS 

Rule improves legibility and increases efficiency of code.
EXAMPLE 

void foo( int x, int y ) {
    x = x + 2; // Violation
    x = x - 2; // Violation
    x = x * 2; // Violation
    x = x / 2; // Violation
    x = x % 2; // Violation
}
REPAIR 

void foo( int x, int y ) {
    x += 2;    // OK
    x -= 2;    // OK
    x *= 2;    // OK
    x /= 2;    // OK
    x %= 2;    // OK
}
