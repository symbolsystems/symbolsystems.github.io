The standard header file <fenv.h> shall not be used [CODSTA-109_b-3]

DESCRIPTION 

"The standard header file <fenv.h> shall not be #included, and none of the
features that are specified as being provided by <fenv.h> shall be used."
[MISRA C:2012 Rule 21.12 with Amendment 3]

The rule detects when a file includes the 'fenv.h' or 'cfenv' header.
SINCE 

2022.2
BENEFITS 

This rule helps you prevent undefined behaviours.
EXAMPLE 

#include <fenv.h>                                  // Violation

void foo ( float x, float y )
{
    float z;

    feclearexcept ( FE_DIVBYZERO );

    z = x / y;

    if ( fetestexcept ( FE_DIVBYZERO ) )
    {
    }
}
REPAIR 

Do not include the <fenv.h> header file.
