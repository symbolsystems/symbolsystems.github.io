Hungarian notation for pointer [NAMING-HN-34-3]

DESCRIPTION 

For pointers use "p" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

const int kiMAX=10;

int *iGrade;                     // Violation

void foo()
{
    int *iMark;                  // Violation
    char * cLetter = 0;          // Violation
}
REPAIR 

const int kiMAX=10;

int *piGrade;                    // OK

void foo()
{
    int *piMark;                 // OK
    char * pcLetter = 0;         // OK
}
