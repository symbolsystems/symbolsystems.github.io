Hungarian notation for ifstream type variables and parameters [NAMING-HN-21-3]

DESCRIPTION 

For ifstream type variables and parameters use "if" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

#include <fstream>
using namespace std;

ifstream NameFile;          // Violation

void foo( ) {
    ifstream Filename;      // Violation
    ifstream *pFilename;    // Violation
}
REPAIR 

#include <fstream>
using namespace std;

ifstream ifNameFile;        // OK 

void foo( ) {
    ifstream ifFilename;    // OK
    ifstream *pifFilename;  // OK
}
