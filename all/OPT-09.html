Remove unnecessary '== true' [OPT-09-5]

DESCRIPTION 

This rule reports a violation if boolean constant 'true' is used as operand 
of equality operator '=='.
EXCEPTIONS 

Violation is not reported if equality operator '==' is used inside
'if' statement condition.
BENEFITS 

The rule increases legibility and efficiency of code.
EXAMPLE 

bool isPositive(int a);

bool foo() 
{
    return isPositive(5) == true;  // Violation
}
REPAIR 

bool isPositive(int a);

bool foo1() 
{
    return isPositive(5);        // OK
}
