There shall be no white spaces between the 'return' statement or the 'sizeof' operator and the following opening parenthesis [FORMAT-23-3]

DESCRIPTION 

This rule detects white spaces between the 'return' statement or the 'sizeof'
operator and its operand enclosed in parentheses. For the 'return' statement,
this is checked only if the whole operand is in parentheses.

See also: FORMAT-24, FORMAT-25
NOTES 

The following white spaces are detected: spaces, tabs, new-line characters,
and comments.
BENEFITS 

This rule helps you improve the readability of the code.
EXAMPLE 

int f1(int x)
{
   sizeof (int);    // Violation
   return (x + 1);  // Violation
}
REPAIR 

int f1(int x)
{
   sizeof(int);     // OK
   return(x + 1);   // OK
}
