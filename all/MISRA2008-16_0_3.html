#undef shall not be used [MISRA2008-16_0_3-2]

DESCRIPTION 

"#undef should not normally be needed. Its use can lead to confusion with
respect to the existence or meaning of a macro when it is used in the code."
[MISRA-C:2004]

This rule detects when the #undef directive is used in code.
BENEFITS 

This rule helps you improve code readability and maintainability.
EXAMPLE 

#define SUM(A,B) ((A)+(B))
// ...
#undef SUM                    // Violation
REPAIR 

You can fix your code by removing the #undef directive.
