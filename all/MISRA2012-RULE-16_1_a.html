A switch statement shall only contain switch labels and switch clauses, and no other code [MISRA2012-RULE-16_1_a-2]

DESCRIPTION 

"A switch statement shall only contain switch labels
and switch clauses, and no other code.
Switch label is either a case label or default label.
Switch clause is either a case clause or a default clause.
Case clause is the code between any two switch labels.
Default clause is the code between the default label
and the end of the switch statement."
BENEFITS 

The rule enforces a simple and consistent structure on to the switch statement.
EXAMPLE 

void foo(int a, int b)
{
    switch(a)
    {
        int c = 0;  /* Violation */
    case 1:
        break;
    case 2:
        break;
   
    default:
        break;
    }
    
    switch(b)
      a = 3;        /* Violation */
}
REPAIR 

void foo(int a,int b)
{
    int c = 0;    /* OK */
    switch(a)
    {
    case 1:
        break;
    case 2:
        break;
   
    default:
        break;
    }
    
    a = 3;        /* OK */
}
