Place left parenthesis directly after function name [FORMAT-30-3]

DESCRIPTION 

"Always write the left parenthesis directly after a function name."
BENEFITS 

Rule improves readability of code.
EXAMPLE 

void 
foo () {	// Violation
}
REPAIR 

void
foo() {		// OK
}
REFERENCES
