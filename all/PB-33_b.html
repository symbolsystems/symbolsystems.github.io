Avoid implicit type conversions in comparisons involving enum types [PB-33_b-3]

DESCRIPTION 

Rule reports violation if enumerators of different enum types are compared 
or if enumerator is compared with non enumerator.
SINCE 

v9.6
BENEFITS 

Prevents depending on the value of enum constant which can change.
EXAMPLE 

enum Day {
    Mon, Tue, Wen, Thu, Fri
};

enum WeekDay {
    Sat, Sun
};

int foo() {
    enum Day a_day;
    if ( a_day == Sun ) return 1;   // Violation
    return 0;
}
REPAIR 

enum Day {
    Mon, Tue, Wen, Thu, Fri, Sat, Sun
};

int foo() {
    enum Day a_day;
    if ( a_day == Sun ) return 1;   // OK
    return 0;
}
REFERENCES 

1. ISO/DIS 26262
   point 8.4.4
