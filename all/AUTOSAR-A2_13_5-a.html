Hexadecimal constants will be represented using all uppercase letters [AUTOSAR-A2_13_5-a-4]

DESCRIPTION 

Hexadecimal constants will be represented using all uppercase letters.
See also: CODSTA-51
SINCE 

v7.1
BENEFITS 

Improved readability and maintenance.
EXAMPLE 

int i = 0x3fff;   // Violation
REPAIR 

int i = 0x3FFF;   // OK
REFERENCES 

1. AUTOSAR C++14 Coding Guidelines
   A2-13-5. Hexadecimal constants should be upper case

2. JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
   Chapter 4.18 Constants, AV Rule 150
