Begin all long integer variables with 'li' [NAMING-21-3]

DESCRIPTION 

All long integer variables should begin with 'li'.
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

void foo() 
{
   long int Var;	// Violation
}
REPAIR 

void foo()
{
   long int liVar;	// OK
}
