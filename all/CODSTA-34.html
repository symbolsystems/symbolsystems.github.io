Use a typedef to simplify program syntax when declaring function pointers [CODSTA-34-3]

DESCRIPTION 

"Use a typedef to simplify program syntax when declaring function pointers."
BENEFITS 

"Another reason to use typedef is that the readability of the code is improved.
If pointers to functions are used, the resulting code can be almost unreadable.
By making a type declaration for the function type, this is avoided."
EXAMPLE 

void (*p)();                 // Violation
REPAIR 

typedef void (*PTF)();

PTF p;              // OK
