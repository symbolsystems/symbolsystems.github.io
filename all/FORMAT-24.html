There shall be no spaces after the opening '(' and before closing ')' parenthesis that enclose the operand of the 'return' statement or the 'sizeof' operator [FORMAT-24-3]

DESCRIPTION 

This rule detects when a space or tab is used after the opening '(' or before
closing ')' parenthesis that follow the 'return' or 'sizeof' keyword.

See also: FORMAT-23, FORMAT-25
EXCEPTIONS 

Spaces are allowed after the opening parenthesis when they are followed
by a comment or a backslash on the same line.

Spaces are allowed before the closing parenthesis if they are not
preceded by another token placed on the same line.

int f1(int x)
{
  return (        // OK
           x + 2
         );       // OK
}
BENEFITS 

This rule helps you improve the readability of the code.
EXAMPLE 

int f1( int x )
{
   unsigned int ui1 = sizeof( int);            // Violation
   unsigned int ui2 = sizeof(unsigned int );   // Violation
   return( x + 1 );                            // Violation
}
REPAIR 

int f1( int x )
{
   unsigned int ui1 = sizeof(int);             // OK
   unsigned int ui2 = sizeof(unsigned int);    // OK
   return(x + 1);                              // OK
}
