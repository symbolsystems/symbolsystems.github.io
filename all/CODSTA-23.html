All 'if' statements should have an 'else' clause [CODSTA-23-3]

DESCRIPTION 

The rule reports a violation on 'if' statement without an 'else' statement.
See also: MISRA2004-14_10
BENEFITS 

Rule ensures proper data flow and improves readability and maintainability.
EXAMPLE 

int foo(int a, int b) {
    if(a > b){          // Violation
        return a - b;
    }

    if(a < b){          // Violation
    	return b - a;
    }

    if(a == b)          // Violation
        return 1;
}
REPAIR 

int foo(int a, int b) {
    if(a > b){          // OK
        return a - b;
    } else if(a < b){   // OK
    	return b - a;
    } else {
    	return 1;
    }
}
