Hungarian notation for int types [NAMING-HN-30-3]

DESCRIPTION 

For int variables and parameters use "n" prefix.
SINCE 

v7.0
BENEFITS 

Improves readability and maintainability.
EXAMPLE 

int g_nmyVar;                                  // Violation
int g_MyVariable;                              // Violation
static const int g_skMyVariable = 0;           // Violation
class A {
    int m_MyVariable;                          // Violation
    static const int m_skMyVariable;           // Violation
    void foo( int MyParam ) {                  // Violation
        int MyVariable;                        // Violation
        static const int skMyVariable = 0;     // Violation
    }
};
REPAIR 

int g_nMyVar;                                 // OK
int g_nMyVariable;                            // OK
static const int g_sknMyVariable = 0;         // OK
class A {
    int m_nMyVariable;                        // OK
    static const int m_sknMyVariable;         // OK
    void foo( int nMyParam ) {                // OK
        int nMyVariable;                      // OK
        static const int sknMyVariable = 0;   // OK
    }
};
