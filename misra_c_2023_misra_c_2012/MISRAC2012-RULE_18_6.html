MISRAC2012-RULE_18_6 
Rule 18.6 (Required) The address of an object with automatic or thread-local storage shall not be copied to another object that persists after the first object has ceased to exist
RULES

The address of an object with automatic storage shall not be returned from a function [MISRAC2012-RULE_18_6-a-2]
The address of an object with automatic storage shall not be assigned to another object that may persist after the first object has ceased to exist [MISRAC2012-RULE_18_6-b-2]
The address of an object with thread storage duration shall not be assigned to an object with static storage duration [MISRAC2012-RULE_18_6-c-2]
