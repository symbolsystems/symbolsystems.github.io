MISRAC2012-RULE_1_5 
Rule 1.5 (Required) Obsolescent language features shall not be used
RULES

The static storage class specifier shall be used in definitions and declarations of objects and functions that have internal linkage [MISRAC2012-RULE_1_5-a-2]
Storage type modifiers shall be associated with the type, not the variable or the function [MISRAC2012-RULE_1_5-b-2]
Function types shall be in prototype form [MISRAC2012-RULE_1_5-c-2]
Do not use the macro ATOMIC_VAR_INIT [MISRAC2012-RULE_1_5-d-2]
Do not use the 'bool', 'false' or 'true' identifiers in the #undef directive [MISRAC2012-RULE_1_5-e-2]
Do not use the ungetc function on a stream with the file position indicator zero [MISRAC2012-RULE_1_5-f-2]
Do not pass 0 value as a size argument to the realloc function [MISRAC2012-RULE_1_5-g-2]
