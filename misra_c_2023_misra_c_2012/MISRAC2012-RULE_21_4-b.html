The standard header files <setjmp.h> or <csetjmp> shall not be used [MISRAC2012-RULE_21_4-b-2]

DESCRIPTION 

"The standard header file <setjmp.h> shall not be #included, and none of the
features that are specified as being provided by <setjmp.h> shall be used."
[MISRA C:2012 Rule 21.4 with Amendment 3]

"'setjmp' and 'longjmp' allow the normal function call mechanisms to be
bypassed. Their use may lead to undefined and unspecified behaviour."
[MISRA C:2012]

The rule detects when the <setjmp.h> or <csetjmp> header is included.
SINCE 

v9.5
BENEFITS 

The rule prevents unspecified and undefined behavior.
EXAMPLE 

#include <setjmp.h>               // Violation
REPAIR 

Do not use the <setjmp.h> header file.
