Do not include <locale.h> header [MISRA-121_b-3]

DESCRIPTION 

The header <locale.h> shall not be used.
BENEFITS 

Prevents changing the locate from the standard C locate.
EXAMPLE 

#include <locale.h>  /* Violation */
void foo(void)
{
  /* Set the locale back to the default environment */
  setlocale(LC_ALL, "C");  
}
REPAIR 

Do not include <locale.h> header.
REFERENCES 

1. AUTOSAR C++14 Coding Guidelines
   A18-0-3. The library <clocale> (locale.h) and the setlocale function shall
   not be used

2. JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
   Chapter 4.5 Libraries, AV Rule 19

3. Origin: Misra Guidelines - Rule 121
