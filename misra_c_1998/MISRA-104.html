Do not use non-constant pointers to functions [MISRA-104-3]

DESCRIPTION 

Non-constant pointers to functions shall not be used.
See also: CODSTA-CPP-03, CODSTA-CPP-38, CODSTA-CPP-43, CODSTA-CPP-44, MISRA2004-16_7, OPT-21
BENEFITS 

Prevents using non-constant pointers to functions.
EXAMPLE 

void foo( );
typedef void (*PFUNC) (void);
const PFUNC pf = foo;

void bar( ) {
    PFUNC f1 = foo;         /* Violation */
}
REPAIR 

void foo( );
typedef void (*PFUNC) (void);
const PFUNC pf = foo; 

void bar( ) {
    const PFUNC f1 = foo;   /* OK */ 
}
