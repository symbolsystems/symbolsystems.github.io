APSC_DV-002400 
APSC_DV-002400 The application must restrict the ability to launch Denial of Service (DoS) attacks against itself or other information systems.
RULES

Exclude unsanitized user input from format strings [APSC_DV-002400-a-2]
The execution of a function registered with 'std::atexit()' or 'std::at_quick_exit()' should not exit via an exception [APSC_DV-002400-b-2]
Avoid using the 'vfork()' function [APSC_DV-002400-c-2]
Avoid using thread-unsafe functions [APSC_DV-002400-d-2]
