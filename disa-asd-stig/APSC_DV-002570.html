APSC_DV-002570 
APSC_DV-002570 The application must generate error messages that provide information necessary for corrective actions without revealing information that could be exploited by adversaries.
RULES

Avoid passing sensitive data to functions that write to log files [APSC_DV-002570-a-2]
Do not print potentially sensitive information, resulting from an application error into exception messages [APSC_DV-002570-b-2]
