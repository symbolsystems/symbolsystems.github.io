APSC_DV-002290 
APSC_DV-002290 The application must use the Federal Information Processing Standard (FIPS) 140-2-validated cryptographic modules and random number generator if the application implements encryption, key exchange, digital signature, and hash functionality.
RULES

Do not use the rand() function for generating pseudorandom numbers [APSC_DV-002290-a-2]
Properly seed pseudorandom number generators [APSC_DV-002290-b-2]
The 'random_shuffle' identifier should not be used [APSC_DV-002290-c-2]
The random number generator functions 'rand()' and 'srand()' should not be used [APSC_DV-002290-d-2]
