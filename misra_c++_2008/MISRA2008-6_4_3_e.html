A switch statement shall be a well-formed switch statement [MISRA2008-6_4_3_e-2]

DESCRIPTION 

Every switch statement shall have at least one case.
See also: OPT-21, OPT-22, CODSTA-54
BENEFITS 

Provides maintainability of 'switch' statement.
EXAMPLE 

void foo(int i) 
{
    switch(i)      /* Violation */ 
    {
        default:
            ;
    }
}
REPAIR 

void foo(int i) 
{
    switch(i)      /* OK */ 
    {
      case 1:
      {
      }
      default:
            ;
    }
}
