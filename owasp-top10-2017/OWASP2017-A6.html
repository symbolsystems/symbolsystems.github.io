OWASP2017-A6 
OWASP A6 Security Misconfiguration
RULES

Where multiple handlers are provided in a single try-catch statement or function-try-block for a derived class and some or all of its bases, the handlers shall be ordered most-derived to base class [OWASP2017-A6-a-3]
Do not leave 'catch' blocks empty [OWASP2017-A6-b-3]
Properly use errno value [OWASP2017-A6-c-2]
