CERT_C-EXP30 
CERT-EXP30_C (RULE) Do not depend on the order of evaluation for side effects
RULES

The value of an expression shall be the same under any order of evaluation that the standard permits [CERT_C-EXP30-a-2]
Don't write code that depends on the order of evaluation of function arguments [CERT_C-EXP30-b-2]
Don't write code that depends on the order of evaluation of function designator and function arguments [CERT_C-EXP30-c-2]
Don't write code that depends on the order of evaluation of expression that involves a function call [CERT_C-EXP30-d-2]
