A function should not call more than 7 different functions [METRICS-37-3]

DESCRIPTION 

The rule reports a violation if a function contains more than 7 calls
of different functions. All calls of the same function are counted as 1.
SINCE 

v7.3
EXAMPLE 

void goo1(){}
void goo2(){}
void goo3(){}
void goo4(){}
void goo5(){}
void goo6(){}
void goo7(){}
void goo8(){}

void foo2()     /* Violation */
{
    goo1();
    goo2();
    goo3();
    goo4();
    goo5();
    goo6();
    goo7();
    goo8();
}
REPAIR 

void goo1(){}
void goo2(){}
void goo3(){}
void goo4(){}
void goo5(){}
void goo6(){}
void goo7(){}
void goo8(){}

void foo1()    /* OK */
{
    goo1();
    goo2();
    goo3();
    goo4();
}

void foo2()    /* OK */
{
    foo1();
    goo5();
    goo6();
    goo7();
    goo8();
}
