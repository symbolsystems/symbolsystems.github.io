The protected section of a class will be declared before the private section [JSF-057_b-3]

DESCRIPTION 

Protected section must be before private section.
See also: CODSTA-CPP-45, CODSTA-CPP-46, CODSTA-CPP-48, CODSTA-CPP-49
BENEFITS 

Rule improves readability and maintainability.
EXAMPLE 

class Test {	// Violation
public:
    void foo1( );
private:
    void foo3( );
protected:
    void foo2( );
};
REPAIR 

class Test {	// OK
public:
    void foo1( );
protected:
    void foo2( );
private:
    void foo3( );
};
