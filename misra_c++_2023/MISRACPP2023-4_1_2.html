Rule 4.1.2 (Advisory) Deprecated features should not be used
RULES

Do not include headers deprecated in the C++17 standard [MISRACPP2023-4_1_2-a-4]
The 'uncaught_exception' function deprecated in the C++17 standard should not be used [MISRACPP2023-4_1_2-b-4]
Do not use throw exception specifications [MISRACPP2023-4_1_2-c-4]
