MISRACPP2023-18_5_2 
Rule 18.5.2 (Advisory) Program-terminating functions should not be used
RULES

The 'abort()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [MISRACPP2023-18_5_2-a-4]
The 'exit()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [MISRACPP2023-18_5_2-b-4]
The 'quick_exit()' and '_Exit()' functions from the 'stdlib.h' or 'cstdlib' library shall not be used [MISRACPP2023-18_5_2-c-4]
