MISRACPP2023-21_6_2 
Rule 21.6.2 (Required) Dynamic memory shall be managed automatically
RULES

Dynamic heap memory allocation shall not be used [MISRACPP2023-21_6_2-a-2]
Any member function named 'allocate' or 'deallocate' enclosed by namespace 'std' should not be used [MISRACPP2023-21_6_2-b-2]
The 'std::unique_ptr::release' function should not be used [MISRACPP2023-21_6_2-c-2]
