MISRACPP2023-6_4_1 
Rule 6.4.1 (Required) A variable declared in an inner scope shall not hide a variable declared in an outer scope
RULES

Identifier declared in a local or function prototype scope shall not hide an identifier declared in a global or namespace scope [MISRACPP2023-6_4_1-a-2]
Identifiers declared in an inner local scope should not hide identifiers declared in an outer local scope [MISRACPP2023-6_4_1-b-2]
Identifiers declared in a local scope should not hide identifiers declared in a class scope [MISRACPP2023-6_4_1-c-2]
Identifiers declared in a class scope should not hide identifiers declared in a global or namespace scope [MISRACPP2023-6_4_1-d-2]
Identifiers declared in an inner class scope should not hide identifiers declared in outer class scope [MISRACPP2023-6_4_1-e-2]
Member variable names in derived classes should not be the same as member variable names in base classes [MISRACPP2023-6_4_1-f-2]
