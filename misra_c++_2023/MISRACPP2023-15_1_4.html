MISRACPP2023-15_1_4 
Rule 15.1.4 (Advisory) All direct, non-static data members of a class should be initialized before the class object is accessible
RULES

All direct, non-static data members of a non-aggregate class should be initialized at the top of a constructor body [MISRACPP2023-15_1_4-a-4]
All direct, non-static data members of an aggregate class should be initialized as soon as the object is created [MISRACPP2023-15_1_4-b-4]
