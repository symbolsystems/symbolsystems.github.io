MISRACPP2023-6_2_2 
Rule 6.2.2 (Required) All declarations of a variable or function shall have the same type
RULES

All declarations of an object or function shall have compatible types [MISRACPP2023-6_2_2-a-2]
If a function is declared with an exception-specification, then all declarations of the same function (in other translation units) shall be declared with the same set of type-ids [MISRACPP2023-6_2_2-b-2]
