AUTOSAR A27-0-4 C-style strings shall not be used
RULES

Don't use unsafe C functions that do write to range-unchecked buffers [AUTOSAR-A27_0_4-a-2]
Avoid using unsafe string functions that do not check bounds [AUTOSAR-A27_0_4-b-2]
Do not use the 'char' buffer to store input from 'std::cin' [AUTOSAR-A27_0_4-c-2]
C-style strings shall not be used [AUTOSAR-A27_0_4-d-2]
Do not use the 'c_str()' and 'data()' functions for string objects [AUTOSAR-A27_0_4-e-2]
