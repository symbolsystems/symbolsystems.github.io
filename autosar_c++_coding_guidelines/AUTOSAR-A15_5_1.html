AUTOSAR-A15_5_1 
AUTOSAR A15-5-1 All user-provided class destructors, deallocation functions, move constructors, move assignment operators and swap functions shall not exit with an exception. A noexcept exception specification shall be added to these functions as appropriate
RULES

Never allow an exception to be thrown from a destructor, deallocation, and swap [AUTOSAR-A15_5_1-a-2]
All user-provided move constructors and move assignment operators shall not exit with an exception [AUTOSAR-A15_5_1-b-2]
