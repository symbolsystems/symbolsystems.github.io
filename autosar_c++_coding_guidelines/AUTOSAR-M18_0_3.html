AUTOSAR M18-0-3 The library functions abort, exit, getenv and system from library <cstdlib> shall not be used
RULES

The 'abort()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [AUTOSAR-M18_0_3-a-2]
The 'exit()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [AUTOSAR-M18_0_3-b-2]
The 'system()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [AUTOSAR-M18_0_3-c-2]
The 'getenv()' function from the 'stdlib.h' or 'cstdlib' library shall not be used [AUTOSAR-M18_0_3-d-2]
