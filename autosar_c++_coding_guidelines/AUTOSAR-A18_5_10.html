AUTOSAR-A18_5_10 
AUTOSAR A18-5-10 Placement new shall be used only with properly aligned pointers to sufficient storage capacity
RULES

Do not pass a pointer that has insufficient storage capacity or that is not suitably aligned for the object being constructed to placement 'new' [AUTOSAR-A18_5_10-a-2]
An overhead should be used when an array of objects is passed to the placement 'new' allocation function [AUTOSAR-A18_5_10-b-2]
