AUTOSAR-A18_5_5 
AUTOSAR A18-5-5 Memory management functions shall ensure the following: (a) deterministic behavior resulting with the existence of worst-case execution time, (b) avoiding memory fragmentation, (c) avoid running out of memory, (d) avoiding mismatched allocations or deallocations, (e) no dependence on non-deterministic calls to kernel
RULES

Use the same form in corresponding calls to new/malloc and delete/free [AUTOSAR-A18_5_5-a-2]
Freed memory shouldn't be accessed under any circumstances [AUTOSAR-A18_5_5-b-2]
Properly define new handlers [AUTOSAR-A18_5_5-c-2]
