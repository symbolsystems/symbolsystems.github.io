Do not use the volatile keyword [AUTOSAR-A2_11_1-a-2]

DESCRIPTION 

Do not use the volatile keyword.
See also: MISRA2004-12_2_f
BENEFITS 

Rule prevents evaluation of expression dependent on compiler version.
EXAMPLE 

void foo1() {
  volatile int v;  /* Violation */
}
REPAIR 

Do not use the volatile keyword.
REFERENCES 

1. AUTOSAR C++14 Coding Guidelines
   A2-11-1. Volatile keyword shall not be used

2. JOINT STRIKE FIGHTER, AIR VEHICLE, C++ CODING STANDARDS
   Chapter 4.25 Expressions, AV Rule 205

3. Origin: Misra Guidelines - Rule 46

4. SEI CERT C Coding Standard
   CON02-C. Do not use volatile as a synchronization primitive
