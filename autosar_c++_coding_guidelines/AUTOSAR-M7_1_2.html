AUTOSAR-M7_1_2 
AUTOSAR M7-1-2 A pointer or reference parameter in a function shall be declared as pointer to const or reference to const if the corresponding object is not modified
RULES

Declare a type of parameter as typedef to pointer to const if the pointer is not used to modify the addressed object [AUTOSAR-M7_1_2-a-2]
A pointer parameter in a function prototype should be declared as pointer to const if the pointer is not used to modify the addressed object [AUTOSAR-M7_1_2-b-2]
Declare reference parameters as const references whenever possible [AUTOSAR-M7_1_2-c-2]
