AUTOSAR M0-1-1 A project shall not contain unreachable code
RULES

There shall be no unreachable code in "if/else/while/for" block [AUTOSAR-M0_1_1-a-2]
There shall be no unreachable code after 'return', 'break', 'continue', 'goto', 'throw' statements, and after calls to functions with the 'noreturn' attribute [AUTOSAR-M0_1_1-b-2]
There shall be no unreachable code in "else" block [AUTOSAR-M0_1_1-c-2]
There shall be no unreachable code after "if" or "switch" statement inside while/for/do...while loop [AUTOSAR-M0_1_1-d-2]
There shall be no unreachable code after 'if' or 'switch' statement [AUTOSAR-M0_1_1-e-2]
There shall be no unreachable code in 'for' loop [AUTOSAR-M0_1_1-f-2]
There shall be no unreachable code in switch statement [AUTOSAR-M0_1_1-g-2]
