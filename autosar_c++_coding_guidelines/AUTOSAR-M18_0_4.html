AUTOSAR M18-0-4 The time handling functions of library <ctime> shall not be used
RULES

The standard header files <time.h> or <ctime> shall not be used [AUTOSAR-M18_0_4-a-2]
The time handling functions and macros of the library <time.h> shall not be used [AUTOSAR-M18_0_4-b-2]
The types defined in the library <time.h> shall not be used [AUTOSAR-M18_0_4-c-2]
