AUTOSAR M5-0-7 There shall be no explicit floating-integral conversions of a cvalue expression
RULES

The value of a complex expression of floating type should not be cast to an integer type [AUTOSAR-M5_0_7-a-2]
The value of a complex expression of integer type shall only be cast to a type of the same signedness that is no wider than the underlying type of the expression [AUTOSAR-M5_0_7-b-2]
