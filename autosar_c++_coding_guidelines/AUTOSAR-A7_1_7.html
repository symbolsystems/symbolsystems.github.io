AUTOSAR-A7_1_7 
AUTOSAR A7-1-7 Each expression statement and identifier declaration shall be placed on a separate line
RULES

Only one statement shall be allowed per line [AUTOSAR-A7_1_7-a-2]
Multiple variable declarations shall not be allowed on the same line [AUTOSAR-A7_1_7-b-2]
Each variable should be declared in a separate declaration statement [AUTOSAR-A7_1_7-c-2]
