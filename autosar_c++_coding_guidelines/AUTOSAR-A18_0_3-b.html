The 'setlocale' function should not be used [AUTOSAR-A18_0_3-b-2]

DESCRIPTION 

"A call to the setlocale function introduces a data race with
other calls to setlocale function." [AUTOSAR A18-0-3]

The rule detects when the global function with the name 'setlocale'
is called or when its address is taken.
BENEFITS 

This rule helps you avoid data races.
EXAMPLE 

#include <locale.h>
void foo(void)
{
    setlocale(LC_ALL, "C");  // Violation
}
REPAIR 

Do not use the 'setlocale' function.
