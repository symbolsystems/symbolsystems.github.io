AUTOSAR-A5_1_4 
AUTOSAR A5-1-4 A lambda expression object shall not outlive any of its reference-captured objects
RULES

Never return lambdas that capture local objects by reference [AUTOSAR-A5_1_4-a-2]
Never capture local objects from an outer lambda by reference [AUTOSAR-A5_1_4-b-2]
The lambda that captures local objects by reference should not be assigned to the variable with a greater lifetime [AUTOSAR-A5_1_4-c-2]
