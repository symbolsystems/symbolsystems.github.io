AUTOSAR-A6_5_4 
AUTOSAR A6-5-4 For-init-statement and expression should not perform actions other than loop-counter initialization and modification
RULES

The initialization expression in a for loop will perform no actions other than to initialize the value of a single for loop parameter [AUTOSAR-A6_5_4-a-4]
The increment expression in a for loop will perform no action other than to change a single loop parameter to the next value for the loop [AUTOSAR-A6_5_4-b-4]
