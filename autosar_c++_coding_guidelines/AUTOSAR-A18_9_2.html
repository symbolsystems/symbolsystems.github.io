AUTOSAR A18-9-2 Forwarding values to other functions shall be done via: (1) std::move if the value is an rvalue reference, (2) std::forward if the value is forwarding reference
RULES

The 'std::forward' function should be used to forward forwarding references [AUTOSAR-A18_9_2-a-2]
The 'std::move' function should be used to forward rvalue references [AUTOSAR-A18_9_2-b-2]
