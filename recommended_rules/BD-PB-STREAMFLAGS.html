Suspicious setting of stream flags [BD-PB-STREAMFLAGS-3]

DESCRIPTION 
This rule detects when stream flags are set using a suspicious value. Stream flags
should always be set using builtin enums (e.g., std::ios::hex).

The following methods can be used to set stream flags:
- fmtflags setf(fmtflags flags);
- fmtflags setf(fmtflags flags, fmtflags mask);
- void unsetf(fmtflags mask);
- fmtflags flags(fmtflags flags);
SINCE 
v10.1
NOTES 
N/A
SECURITY RELEVANCE 
N/A
PARAMETERS 
N/A
BENEFITS 
Setting the stream format with suspicious value can indicate a possible bug in the code.
DRAWBACKS 
N/A
EXAMPLE 
Here is an example that triggers a violation:

#include <iostream>

using namespace std;

void bad(iostream& stream) {
    cout.setf((ios_base::fmtflags) 1); // VIOLATION
    cout.setf(ios_base::hex | (ios_base::fmtflags) 0x08); // VIOLATION
    ios_base::fmtflags flags = (ios_base::fmtflags) 123;
    stream.unsetf(flags); // VIOLATION
    ios_base::fmtflags flags2 = flags | ios_base::uppercase;
    stream.flags(flags2); // VIOLATION
}
REPAIR 

#include <iostream>

using namespace std;

void ok(iostream& stream) {
    cout.setf(ios_base::hex);
    cout.setf(ios_base::hex | ios_base::uppercase);
    ios_base::fmtflags flags = ios_base::hex;
    stream.unsetf(flags);
    ios_base::fmtflags flags2 = flags | ios_base::uppercase;
    stream.flags(flags2);
}
